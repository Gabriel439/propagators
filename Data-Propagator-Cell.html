data Cell s a Source

A Cell contains "information about a value" rather than a value per se.

Requiring the value of a Cell will return the value of the cell if present.

watch :: Cell s a -> (a -> ST s ()) -> ST s () Source

Watching a Cell sets up a callback that will be notified if the cell changes.

with :: Cell s a -> (a -> ST s ()) -> ST s () Source

with will read the current value of a Cell and do something with that result if it is known. If the Cell is currently empty, this will do nothing. Unlike watch it does not install a handler.

watch2 :: Cell s a -> Cell s b -> (a -> b -> ST s ()) -> ST s () Source

watch2 will watch two cells. When they both have some information the supplied callback will fire at least once. It will continue to fire each time they get more information from then out.
